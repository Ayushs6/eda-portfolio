# world-development-indicators-eda

# Exploratory Data Analysis - World Development Indicators (WDI)

## Overview
Exploratory data analysis and visualizations on a subset of the World Bank WDI dataset (2000–2022).

## Countries
Brazil, Canada, China, Germany, India, Japan, South Africa, United States

## Indicators
- Access to Electricity (%)
- Health Expenditure per capita (US$)
- GDP per capita (current US$)
- Life Expectancy (years)
- Urban Population (%)

## Files
- FINAL REVISON_EDA_Project.ipynb — full analysis + plots
- Presentation.pptx — summary slides

## How to run
Open the notebook in Jupyter/Colab and run cells top-to-bottom.
